# GitHub-Style Alerts Demo

All alerts use **SVG icons** with transparent (ghost style) effect.

## Note (Azure)

> [!NOTE]
> Useful information that users should know, even when skimming content.
> Icon: Info circle with 60% opacity that becomes 90% on hover.

## Tip (Jade)

> [!TIP]
> Helpful advice for doing things better or more easily.
> Icon: Light bulb to represent ideas/tips.

## Important (Violet)

> [!IMPORTANT]
> Key information users need to know to achieve their goal.
> Icon: Message bubble for important communication.

## Warning (Amber)

> [!WARNING]
> Urgent info that needs immediate user attention to avoid problems.
> Icon: Triangle warning sign.

## Caution (Red)

> [!CAUTION]
> Advises about risks or negative outcomes of certain actions.
> Icon: Octagon stop sign for danger.

---

## Features

### SVG Icons with Ghost Style
- **Opacity**: 60% (default), 90% (hover)
- **Fill color**: Follows Pico color palette
- **Smooth transition**: 0.2s ease
- **Responsive**: 16x16px

### Light & Dark Mode
- Light mode: Darker colors (color-600)
- Dark mode: Lighter colors (color-400)
- Auto-adjust with theme switcher

## Multiple Paragraphs

> [!NOTE]
> This is a note with multiple paragraphs.
>
> You can have multiple lines and paragraphs inside alerts.
> They work just like regular blockquotes!
